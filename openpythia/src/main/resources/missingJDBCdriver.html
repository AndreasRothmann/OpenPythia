Missing JDBC driver
Pythia needs JDBC drivers to connect to the Oracle database. There isn't neither such a driver in the class path nor at a place defined by you.
Oracle keeps these driver under a licence that does not allow to include them within Pythia. Because of this, you have to download the driver from Oracle (last seen here) and direct Pythia the path to the downloaded file. Therefore press the button Load JDBC Driver, select the file within the file selection dialog and press OK.
